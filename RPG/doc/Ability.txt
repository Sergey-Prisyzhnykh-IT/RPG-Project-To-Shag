Класс Ability необходим для наследования от него классов Buff и AttackSpell
Имеет поля:
string Name - название способности
int ManaCoast - необходимое кол-во маны на использование способности
int Level - уровень, необходимый для изучения способности
int Value - значение нанечения урона или величина положительного воздействия на персонажа

Имеет конструктор по умолчанию и с параметрами.
В конструктор с параметрами передаются: string name, int manaCast. Заполняются соответствующие поля.

Внутри файла Ability.cs объявленны ещё два класса - Buff и AttackSpell, оба наследованы от Ability
(разделение сделали для логики использования)
