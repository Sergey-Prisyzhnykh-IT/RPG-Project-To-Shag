Класс Hero наследуется от интерфейса IAbility
Имеет поля:
string Name - имя персонажа
int Level - уровень персонажа
int Experience - накопленный опыт персонажа
int ExperienceToLevel - необходимое кол-во опыта для повышения уровня
int SkillPoints - очки, которые можно потратить на изучение способностей
int StatPoints - очки, которые можно потратить на изучение характеристик
int Health - здоровье персонажа
int Mana - запас маны персонажа
int Strength - показатель силы персонажа
int Agility - показатель ловкости персонажа
int Intellect - показатель интеллекта персонажа
int Vitality - показатель выносливости персонажа
List<Ability> Ability - способности персонажа

Класс имеет конструктор по умолчанию, а так же конструктор с параметрами.
В конструктор с параметрами передаётся имя, заполняются поля Name, Level, Experience,
Strength, Agility, Intellect, Vitality. А так же вызывается метод UpdateHero.

Методы:
Метод void UpdateHero не принимает аргументов, необходим для определения 
полей Health, Mana и ExperienceToLevel.

Метод void LevelUp принимает аргумент типа int, необходим для подсчёта 
опыта и повышения уровня(если опыта достаточно).

Методы void StrengthUp, void AgilityUp, void IntellectUp, void VitalityUp необходимы для повышения
характеристик за SkillPoints, аргументы не принимают.

Метод void StudyAbility принимает аргумент типа Ability, необходим для изучения способности.
Если передаваемая способность уже изучена, то ничего не произойдёт.

Метод int Damage возвращает показатель урона (необходим для тестирования).
